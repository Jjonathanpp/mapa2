# Regalo de cumpleaños (mobile-first)

Página web pensada para verse desde celular: scroll vertical con fotos + frases + música de fondo.

## Cómo usarla

1. Reemplazá las fotos en `assets/`:
   - `photo-1.svg`
   - `photo-2.svg`
   - `photo-3.svg`
   - `photo-4.svg`

   Podés usar `.jpg`, `.png` o `.webp`; si cambiás el nombre/extensión, actualizá `index.html`.

2. Reemplazá `assets/song.mp3` por la canción real.

3. Editá los textos de las frases en `index.html` para que coincidan con la letra real.

## Publicarlo desde GitHub (sin servicios externos)

1. Subí este repo a GitHub.
2. Andá a **Settings → Pages**.
3. En **Build and deployment**, elegí:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (o la rama que uses), carpeta `/ (root)`
4. Guardá y esperá 1-2 minutos.
5. Tu URL quedará así:
   - `https://TU-USUARIO.github.io/TU-REPO/`

Eso permite que cualquiera entre al link desde el celular sin instalar nada.
